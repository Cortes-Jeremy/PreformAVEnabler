1. Download **[Latest Version](https://github.com/Cortes-Jeremy/PreformAVEnabler/releases/latest)**
2. Unpack the Zip file
3. Copy (or drag and drop) **AAVBlackrock** into your Wow-Directory\Interface\AddOns
4. Restart WoW
